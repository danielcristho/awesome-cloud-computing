# Tools & Software

> Collections of tools and software to manage cloud infrastructure across different categories and use cases.

## Infrastructure & Management

> Core tools and platforms for provisioning, managing, migrating, and protecting cloud infrastructure.

| Category | Description |
|----------|-------------|
| **Infrastructure as Code** | Tools for managing infrastructure through code. |
| **Multi-cloud Management** | Platforms for managing multiple cloud providers. |
| **Cloud Migration Tools** | Services and tools for migrating workloads to cloud environments. |
| **Disaster Recovery** | Backup and disaster recovery solutions. |

## Development & Deployment

> Tools for building, deploying, and running modern cloud-native applications.

| Category | Description |
|----------|-------------|
| **Containerization** | Container engines, orchestration, and management tools. |
| **Serverless Frameworks** | Tools and frameworks for building serverless applications. |
| **Edge Computing** | Platforms for edge and distributed computing. |

## Operations & Monitoring

> Observability, operations, and cost management tools for cloud workloads.

| Category | Description |
|----------|-------------|
| **Monitoring** | Tools for monitoring cloud services and applications. |
| **Logging** | Solutions for log collection, management, and analysis. |
| **FinOps & Cost Management** | Tools and practices for cloud cost optimization and financial governance. |
