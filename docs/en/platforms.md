# Cloud Platforms

> A list of public, private, and hybrid cloud computing platforms,
> including open-source and commercial offerings.

## Major Public Cloud Providers

> Large-scale public cloud platforms offering managed infrastructure and services.

| Platform | Description | Official Website |
|----------|-------------|------------------|
| **AWS (Amazon Web Services)** | Public cloud platform offering computing, storage, databases, and managed services. | [aws.amazon.com](https://aws.amazon.com/console) |
| **Azure (Microsoft Azure)** | Cloud computing platform providing infrastructure services, application platforms, and hybrid cloud support. | [azure.microsoft.com](https://azure.microsoft.com/en-us) |
| **GCP (Google Cloud Platform)** | Cloud computing services focused on application workloads, data, and analytics. | [cloud.google.com](https://cloud.google.com/?hl=en) |

## Enterprise & Commercial Cloud Providers

> Cloud platforms focused on enterprise workloads and commercial infrastructure offerings.

| Platform | Description | Official Website |
|----------|-------------|------------------|
| **OCI (Oracle Cloud Infrastructure)** | Cloud platform supporting enterprise workloads, databases, and infrastructure services. | [oracle.com/cloud](https://www.oracle.com/cloud) |
| **IBM Cloud** | Cloud platform offering virtual servers, bare metal, and managed services. | [ibm.com/cloud](https://www.ibm.com/cloud) |
| **Alibaba Cloud** | Cloud computing platform providing infrastructure and platform services for enterprise and global workloads. | [alibabacloud.com](https://www.alibabacloud.com) |
| **SAP Cloud Infrastructure** | Cloud infrastructure services designed to support SAP and enterprise workloads. | [sap.com/cloud](https://www.sap.com/products/technology-platform.html) |

## Developer-Focused Cloud Providers

> Cloud providers emphasizing simplicity and infrastructure services for application development.

| Platform | Description | Official Website |
|----------|-------------|------------------|
| **DigitalOcean** | Cloud platform focused on virtual machines and managed services for application deployment. | [digitalocean.com](https://www.digitalocean.com) |
| **Linode** | Cloud hosting provider offering Linux virtual machines and related infrastructure services. | [linode.com](https://www.linode.com) |
| **Vultr** | Cloud provider offering virtual machines, bare metal, and global deployment options. | [vultr.com](https://www.vultr.com) |
| **Hetzner Cloud** | Cloud provider offering virtual machines and infrastructure services, primarily in Europe. | [hetzner.com/cloud](https://www.hetzner.com/cloud) |

## Open-Source Cloud Platforms

> Open-source platforms for building and managing private or hybrid cloud infrastructure.

| Platform | Description | Official Website |
|----------|-------------|------------------|
| **OpenStack** | Open-source software for building private and hybrid cloud infrastructure. | [openstack.org](https://www.openstack.org) |
| **Apache CloudStack** | Open-source software designed to deploy and manage large networks of virtual machines as an Infrastructure as a Service (IaaS) platform. | [cloudstack.apache.org](https://cloudstack.apache.org) |
| **OpenNebula** | Open-source platform for managing virtualized data centers and building private and hybrid clouds. | [opennebula.io](https://opennebula.io) |
| **Eucalyptus** | Open-source Infrastructure as a Service (IaaS) platform designed to build private clouds compatible with Amazon Web Services APIs. | [eucalyptus.cloud](https://www.eucalyptus.cloud) |
