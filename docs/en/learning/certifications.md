# Certifications

> Certification programs and training resources related to cloud computing,
> offered by cloud providers and independent organizations.

## Free Certifications

| Name | Description | Link |
|------|-------------|------|
| **AWS Educate** | A free program by Amazon providing learning resources and training materials for students and educators interested in cloud computing. | [AWS Educate](https://aws.amazon.com/education/awseducate) |
| **Google Cloud Skills Boost** | A learning platform offering labs and courses to help users explore Google Cloud services and concepts. | [Google Cloud Skills Boost](https://www.cloudskillsboost.google) |

## Paid Certifications

| Name | Description | Link |
|------|-------------|------|
| **AWS Certifications** | Certification programs covering a range of roles and skill levels related to Amazon Web Services. | [AWS Certifications](https://aws.amazon.com/certification) |
| **Google Cloud Certification** | Certification paths focused on Google Cloud technologies and roles. | [GCP Certification](https://cloud.google.com/learn/certification?hl=en) |
| **CompTIA Cloud+** | A vendor-neutral certification covering cloud infrastructure, operations, security, and troubleshooting concepts. | [CompTIA Cloud+](https://www.comptia.org/certifications/cloud) |

## Certification Paths

> The following examples group certifications by commonly referenced roles.
> Actual certification requirements and scopes may vary by provider.

### Cloud Practitioner / Fundamentals

- **AWS Certified Cloud Practitioner** - Entry-level certification covering basic AWS concepts
- **Microsoft Azure Fundamentals (AZ-900)** - Introduction to core Azure services and concepts
- **Google Cloud Digital Leader** - Overview of cloud concepts with a business focus

### Solutions Architect

- **AWS Certified Solutions Architect** - Associate and Professional levels
- **Microsoft Azure Solutions Architect Expert** - Design-focused Azure certification
- **Google Professional Cloud Architect** - Architecture-related certification for Google Cloud

### Developer

- **AWS Certified Developer** - Application development on AWS
- **Microsoft Azure Developer Associate** - Application development using Azure services
- **Google Professional Cloud Developer** - Application development on Google Cloud

### DevOps Engineer

- **AWS Certified DevOps Engineer** - DevOps-focused certification on AWS
- **Microsoft Azure DevOps Engineer Expert** - Azure DevOps practices and tools
- **Google Professional Cloud DevOps Engineer** - DevOps practices on Google Cloud

### Security Specialist

- **AWS Certified Security - Specialty** - Security-related AWS services and practices
- **Microsoft Azure Security Engineer** - Security implementation on Azure
- **Google Professional Cloud Security Engineer** - Security design and operations on Google Cloud
